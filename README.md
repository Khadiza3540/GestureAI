# GestureAI - Hand Gesture Recognition System

## Overview

GestureAI is a Computer Vision and Machine Learning project that recognizes hand gestures in real time using a webcam. The system uses MediaPipe for hand landmark detection and Machine Learning algorithms for gesture classification.

The project can be used for gesture-based interaction, sign language recognition, educational applications, and human-computer interaction systems.

## Features

* Real-Time Hand Tracking
* Hand Landmark Detection
* Gesture Classification
* Webcam-Based Recognition
* Machine Learning Prediction
* Fast and Lightweight Processing
* Interactive User Interface

## Technologies Used

* Python
* MediaPipe
* OpenCV
* Scikit-learn
* NumPy
* Jupyter Notebook

## Machine Learning Workflow

1. Collect Hand Gesture Data
2. Detect Hand Landmarks using MediaPipe
3. Extract Landmark Features
4. Train Machine Learning Model
5. Real-Time Gesture Prediction
6. Display Recognition Results
